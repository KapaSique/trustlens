# TrustLens Clean Enterprise Banner

## Goal

Replace the current README banner with a polished, light enterprise visual that presents TrustLens as a credible premium BI product and communicates its core differentiator: deterministic verification of every reported number.

## Composition

- Final canvas: 1200 x 300 pixels.
- Left side: crisp `TrustLens` wordmark and the existing tagline, `The insights agent you can actually trust`.
- Right side: an abstract premium BI scene in which a transparent optical lens inspects a restrained stream of charts, tables, and numeric data.
- Verified data should emerge with one subtle green confirmation signal.
- Keep the composition readable at GitHub README width and leave generous whitespace.

## Visual Language

- Light enterprise aesthetic with a white and cool-gray foundation.
- Graphite typography, restrained cobalt accents, and a small emerald verification accent.
- Precise glass, metal, and data-interface details with soft studio lighting.
- Sophisticated and credible rather than futuristic, playful, or decorative.

## Asset Production

- Generate the visual scene with GPT Image.
- Keep generated text out of the scene to avoid malformed labels.
- Add the product name and tagline during deterministic local compositing so both remain perfectly sharp.
- Export the final project asset as `assets/banner.png`.
- Keep `assets/banner.svg` only if it remains useful as a fallback or source reference; the README continues to use the PNG.

## Constraints

- No neon, cyberpunk styling, dark dominant background, gradients used as the main visual device, visual clutter, watermarks, logos from third parties, or illegible pseudo-text.
- Do not invent product claims or metrics.
- Preserve the existing README structure and banner alt text.

## Validation

- Confirm exact 1200 x 300 dimensions.
- Inspect the final image at full size and a reduced README-like width.
- Verify that title and tagline are spelled correctly and remain legible.
- Confirm the image renders from the README path and that only intended files changed.
